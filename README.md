# AviUtl2-Gemini-Scripts

Geminiで生成したAviUtl2のスクリプト集です。

このリポジトリには様々なスクリプトが含まれていま

## インストール方法

- AviUtl2-compatible scripts
- Open-source and freely available
- Easy to modify and extend
- Experimental and community-friendly

## Installation

1. 必要なスクリプトファイル(*.anm2,*.obj2,*.cam2,*.scn2,*.tra2)をダウンロードする。
2. AviUtl2のスクリプトフォルダーにコピーする。(ファイル名に@が付いていないと動作しないものがあるので注意)
3. AviUtl2を起動しスクリプトを信頼して使用するボタンを押す。
